# TVS-14A2-bridge-adaptor
TVS-14A2 를 브릿지를 이용하여 연결할때 문제점을 보완하기위한 어뎁터 STL 모델링 파일입니다.

하단에 홈이 0º 를 지원합니다

2. 파노모드를 지원하는 연장 어뎁터

![챈투 파노호환 어뎁터](https://github.com/sprtms400/TVS-14A2-bridge-adaptor/assets/26298389/bf667b88-20bf-4b2b-b77e-4e776883e32e)
